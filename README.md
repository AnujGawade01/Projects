# Chat-Messenger
This project is a simple Java-based chat application consisting of two primary files: `ChatMessengerAdmin` and `ChatMessengerClient`.

# Requirements
To run this application, you can use either:
* **Two Computers:** Both connected via a data/LAN cable, with one running the Admin file and the other running the Client file.
* **Single Computer**: A single PC running two separate Command Prompt (CMD) screens.

# How to Run
1. Open your Command Prompt (CMD).
2. Compile & run the Admin Java file first using java command
3. Then Compile & run the CLient Java file using java command to successfully establish connection
